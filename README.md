# Анализ лояльности пользователей Яндекс Афиши

Проект изучает поведение пользователей Яндекс Афиши и факторы, влияющие на повторные покупки билетов.

---

## 1. Данные
- Заказы за период 01.06–31.10.2024 (292 034 строки, 14 столбцов)
- Источник — база данных PostgreSQL, курсы валют — CSV
- Пропуски только в `days_since_prev` (у пользователей с одним заказом)
- Типы данных приведены: даты → datetime, категориальные → category, tickets_count → int16
- Выручка в тенге пересчитана в рубли по дневному курсу → столбец `revenue_rub`
- Выбросы по `revenue_rub` > 99-го перцентиля (> 2 156 руб.) удалены (~1%, 2 884 строки)
---
## 2. Профиль пользователя
Для каждого из 21 815 пользователей собраны признаки:
- Дата первого и последнего заказа
- Устройство, регион, билетный оператор, жанр первого мероприятия
- Средняя выручка с заказа, среднее количество билетов, среднее время между заказами
- Бинарные признаки: `is_two` (2+ заказа), `is_five` (5+ заказов)
---
## 3. Основная статистика
- **21 815** уникальных пользователей
- Средняя выручка с заказа: **524,48 руб.**
- Доля пользователей с 2+ заказами: **61,65%**
- Доля пользователей с 5+ заказами: **28,91%**
---
## 4. Первые заказы и возвраты
- **Жанры:** концерты — главная точка входа (43,7%), но по доле возврата лидируют выставки (64,3%) и театр (63,9%). Спорт — ниже среднего (55,9%)
- **Устройства:** 82,5% приходят с мобильных, десктоп показывает чуть более высокий возврат (64,1% vs 61,2%)
- **Регионы:** Каменевский регион (32,7%) и Североярская область (17,5%) — основные рынки
- **Операторы:** «Билеты без проблем» (23,9%) и «Мой билет» (13,6%) — лидеры
---
## 5. Поведение через выручку и билеты
- Средняя выручка практически не различается: одноразовые — 526 руб., вернувшиеся — 523 руб.
- Сегмент **2–3 билета** показывает максимальную долю возврата (**74,9%**)
- У лояльных пользователей (5+) средний интервал между покупками **9,3 дня** vs 21,0 у группы 2–4 заказа
---
## 6. Корреляционный анализ (phi_k)
- **Сегмент 2–4 заказа:** главные предикторы — интервал между заказами (0,39) и число билетов (0,37)
- **Сегмент 5+ заказов:** ведущий фактор — регион (0,28), далее оператор (0,11)
---
## Выводы и рекомендации
- **Быстрый возврат** — ключ к лояльности: стимулировать повторную покупку в первые 2–3 недели
- **Театр и выставки** дают самый высокий возврат — увеличивать предложение
- **Сегмент 2–3 билета** — самый лояльный, стоит стимулировать парные/семейные покупки
- **Регион** — главный предиктор глубокой лояльности, усилить маркетинг в перспективных регионах
