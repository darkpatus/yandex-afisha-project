# Анализ лояльности пользователей Яндекс Афиши

Проект изучает поведение пользователей Яндекс Афиши и их возвраты на платформу.  

---

## 1. Данные
- Заказы за период 01.06–31.10.2024 (~292k строк, 14 колонок)  
- Пропуски только в days_since_prev (у пользователей с 1 заказом)  
- Типы данных приведены к нужным: даты -> datetime, категориальные -> category, tickets_count -> int16  
- Выручка в тенге переведена в рубли (revenue_rub)  
- Выбросы по revenue_rub > 99 перцентиля удалены (~1%)  

---

## 2. Профиль пользователя
Для каждого пользователя собраны признаки:  
- Дата первого и последнего заказа  
- Устройство, регион, сервис, жанр первого мероприятия  
- Средняя выручка (revenue_rub)  
- Среднее количество билетов  
- Среднее время между заказами  
- Бинарные признаки: is_two (2+ заказа), is_five (5+ заказов)  

---

## 3. Основная статистика
- Средняя выручка с одного заказа: ~xx руб  
- Доля с 2+ заказами: xx%, с 5+ заказами: xx%  
- Статистика по заказам, билетам и дням между покупками в норме, выбросы редкие  

---

## 4. Первые заказы и возвраты
- Тип мероприятия: 2-3 жанра привлекают большинство пользователей  
- Устройство: мобильные/ПК, одна категория лидирует  
- Регион: есть несколько топ-регионов, остальные меньше  
- Сервис: есть лидеры по количеству пользователей  
- Возврат: выше среднего у пользователей с популярными жанрами и в топ-регионах  

---

## 5. Поведение через выручку и билеты
- Средняя выручка у вернувшихся чуть выше, форма распределений схожа с одинарными покупателями  
- Пользователи с большим количеством билетов и выручки могут быть более лояльными  

---

## Вывод
- Есть явные точки входа: популярные жанры, регионы, сервисы, устройства  
- Пользователи из популярных сегментов чаще возвращаются  
- Данные чистые, выбросы учтены, предобработка проведена
